Two choices that mattered

Decision · Day 4
Sentence-boundary streaming over chunked TTS
The LLM emits tokens in <33ms, but Kokoro wants whole sentences for natural prosody. Built a SentenceBoundaryDetector that consumes the snapshot stream and hands completed sentences to a TTSWorker actor. Sentence N+1 inference prefetches while sentence N plays. Result: first spoken word in 104 ms while the rest of the answer is still being generated.
Decision · Day 5
Settings reads only real device state
Privacy is the entire pitch, so the audit screen had to be auditable. Every row pulls live state per render — @Query for meeting and chunk counts, FileManager probes via ModelLocator for bundle integrity, PrivacyMonitor.state for the network row. The Verify button runs an actual probe sequence, not a hardcoded green check. Recorded in ADR-016.
